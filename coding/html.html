HTML Basics

What is HTML?

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It structures the content on the web and allows for elements like text, images, links, and more to be displayed in a browser.

HTML uses "tags" to mark up text, images, links, and other content. For example:

<p>This is a paragraph.</p>

Basic Structure of an HTML Document

A typical HTML document has the following structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <h1>Welcome to HTML Basics</h1>
    <p>This is an introduction to HTML.</p>
</body>
</html>

Make sure to include the <!DOCTYPE html> declaration to define this document as HTML5.

HTML Elements and Tags

HTML elements are defined by tags. Here are some common HTML tags:

  • <h1> ... <h6>: Header tags for headings of different levels.
  • <p>: Paragraph tag for blocks of text.
  • <a>: Anchor tag to create hyperlinks.
  • <img>: Image tag for displaying pictures.

HTML Attributes

Attributes provide additional information about HTML elements. They are defined in the opening tag and come in name/value pairs like this:

<a href="https://www.example.com">Visit Example</a>

In this example, href is an attribute that specifies the destination URL.

Practical Exercises

Try creating your own HTML document using the tags and structure discussed above. Here are a couple of exercises:

  • Create a simple HTML page with a title, a heading, and a paragraph.
  • Add an image and a hyperlink to your page.
  • Experiment with different header tags (<h1> to <h6>) and observe how they appear in the browser.
