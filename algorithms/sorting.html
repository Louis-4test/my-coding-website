Sorting Algorithms

1. Merge Sort

Merge Sort is a divide and conquer algorithm that was invented by John von Neumann in 1945. It divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves.

function mergeSort(arr):
    if length(arr) <= 1:
        return arr
    mid = length(arr) / 2
    left = mergeSort(arr[0...mid])
    right = mergeSort(arr[mid...end])
    return merge(left, right)

2. Bubble Sort

Bubble Sort repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The pass through the list is repeated until the list is sorted.

function bubbleSort(arr):
    n = length(arr)
    for i from 0 to n-1:
        for j from 0 to n-i-2:
            if arr[j] > arr[j+1]:
                swap(arr[j], arr[j+1])
            return arr

3. Quick Sort

Quick Sort is a comparison sorting algorithm that uses divide and conquer principles to sort lists. It selects a pivot element and partitions the other elements into two sub-arrays, according to whether they are less than or greater than the pivot.

function quickSort(arr):
    if length(arr) <= 1:
        return arr
    pivot = arr[length(arr) / 2]
    left = filter(n => n < pivot, arr)
    middle = filter(n => n == pivot, arr)
    right = filter(n => n > pivot, arr)
    return quickSort(left) + middle + quickSort(right)

4. Insertion Sort

Insertion Sort is an efficient algorithm for sorting small data sets. It builds the final sorted array one item at a time, taking each element from the input and finding the correct position in the already sorted part of the array.

function insertionSort(arr):
    for i from 1 to length(arr):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j = j - 1
        arr[j + 1] = key
    return arr

5. Selection Sort

Selection Sort is a simple comparison sorting algorithm. It divides the input array into two parts: the sorted and the unsorted parts. The smallest element from the unsorted part is selected and swapped with the first unsorted element.

function selectionSort(arr):
    n = length(arr)
    for i from 0 to n-1:
        minIndex = i
        for j from i+1 to n:
            if arr[j] < arr[minIndex]:
                minIndex = j
        swap(arr[i], arr[minIndex])
    return arr
